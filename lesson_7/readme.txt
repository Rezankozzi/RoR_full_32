Здесь лежат файлы с выполненным заданием к уроку "Блоки, Proc и lambda"

Файлы с подключаемыми модулями: instance_counter.rb (счетчик количества экземпляров класса); manufacturer.rb (данные о производителе); validation.rb (валидация
                                создаваемого объекта класса); test_method.rb (создание нескольких объектов и связывание их между собой для тестирования 
                                работы программы); папка interfaces - содержит файлы с подключаемыми модулями, описывающими интерфейс пользователя для соот-
                                ветствующего  объекта класса - train_interface.rb, station_interface.rb, route_interface.rb, wagon_interface.rb, menu_interface.rb -  
                                меню интерфейса в виде хэша)

Файлы с описанием классов: train.rb; train_cargo.rb, train_passenger.rb, station.rb; route.rb; wagon.rb, wagon_cargo.rb, wagon_passenger.rb 

Файл с описанием интерфейса: interface.rb

Файл запуска программы: main.rb
